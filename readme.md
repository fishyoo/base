# Note：
- **搜索技巧**
    - 可搜索目录下的文件内容：例如在win7，在base这个目录下搜索所有.doc里的“惊群”，输入搜索后，点击下方的“文件内容”几个字，则可以搜索文件内容。

- **图片压缩注意**
    - WORD文档里的截图图片不要压缩，压缩后.doc 减小的空间不足5%，但是图片清晰度严重变化，看久了伤眼睛(对于这种情况可以尝试将.doc放大或者缩小看看是否会失去模糊)；对于从外部直接粘贴到WORD的图片，图片本身可能很大，所以针对这个图片压缩即可，200分辨率即可。

- **插入我的模版目录**
    - 操作步骤：
引用-->目录-->插入目录-->自定义目录-->“格式”选择“来自模版”
，这样之后，插入的目录便是我的自定义的那套目录。
